# RedStore-athletes-choice-E-commerce-Website
RedStore is a responsive business e-commerce website developed using HTML5, CSS3, and JavaScript. It simulates a modern online shopping platform where users can browse products, view detailed product information, manage a shopping cart, contact the business, and access login and registration features. The website allows users to explore a wide range of fashion products, view detailed product information with image galleries, manage a shopping cart, and contact the business. The website includes multiple pages such as Home, Products, Product Details, Cart, Account (Login/Register), About, and Contact. Key features include a product image gallery with thumbnail switching, a responsive cart layout, and animated login/register form toggling. The UI follows a consistent design language across all pages. Semantic HTML5 was used for structured and accessible markup, while CSS Grid and Flexbox were applied for layouts and alignment. Styles were organized into separate CSS files for base styles, layout, and responsiveness. JavaScript was used for mobile menu toggling, image gallery switching, form validation, and other interactive behaviors.

This project demonstrates the practical implementation of modern frontend web development concepts by building a complete business-oriented e-commerce website. It strengthened skills in responsive design, layout structuring, and interactive UI development, and serves as a strong foundation for future enhancements such as backend integration, database connectivity.
